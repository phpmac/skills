# 标点符号检查器

## 概述

检测并修复代码库中的中文标点符号违规. 强制使用英文标点符号, 提升代码规范性.

## 中文标点符号列表

| 中文标点 | 英文标点 |
|---------|---------|
| `，` | `,` |
| `。` | `.` |
| `：` | `:` |
| `；` | `;` |
| `、` | `,` |
| `｜` | `|` |
| `"` `"` | `"` |
| `'` `'` | `'` |
| `【` `】` | `[` `]` |
| `《》` | `<` `>` |
| `（` `）` | `(` `)` |

**注意**: hookify 规则示例文件中的标点定义除外

## 扫描命令

使用 Grep 工具扫描所有中文标点违规:

```
Grep:
  pattern: "，|。|、|；|：|【|】|（|）|｜"
  output_mode: content
```


## 报告格式

扫描完成后输出:

```markdown
## 标点符号检查结果

| 文件 | 违规数量 | 违规类型 |
|-----|---------|---------|
| path/to/file.md | 3 | ，。： |

**总计**: X 个文件, Y 处违规
```
