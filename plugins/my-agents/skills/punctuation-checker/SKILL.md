# 标点符号检查器

## 原则

**只替换标点符号,保留所有中文内容**

## 中文标点映射

| 中文标点 | 替换为 |
|---------|--------|
| `，` | `,` |
| `。` | `.` |
| `：` | `:` |
| `；` | `;` |
| `、` | `,` |
| `｜` | `|` |
| `【` | `[` |
| `】` | `]` |
| `（` | `(` |
| `）` | `)` |

**注意**: hookify 规则示例文件除外

## 团队模式执行

### 1. 并行扫描

启动多个 agent 并行扫描不同目录,汇总违规结果

### 2. 批量修复

```bash
sed -i '' 's/，/,/g' <files>
sed -i '' 's/。/./g' <files>
sed -i '' 's/、/,/g' <files>
sed -i '' 's/：/:/g' <files>
sed -i '' 's/；/;/g' <files>
sed -i '' 's/【/[/g' <files>
sed -i '' 's/】/]/g' <files>
sed -i '' 's/（/(/g' <files>
sed -i '' 's/）/)/g' <files>
```

### 3. 验证

修复后再次扫描确认零违规

## 扫描命令

```
Grep:
  pattern: "，|。|、|；|：|【|】|（|）|｜"
  output_mode: content
```
