# 标点符号检查

扫描和修复项目中的中文标点符号违规. **只替换标点符号, 保留所有中文内容.**

## 中文标点映射

| 中文标点 | 替换为 |
|---------|--------|
| `，` | `,` |
| `。` | `.` |
| `：` | `:` |
| `；` | `;` |
| `、` | `,` |
| `｜` | `\|` |
| `【` | `[` |
| `】` | `]` |
| `（` | `(` |
| `）` | `)` |

**例外**: `plugins/hookify/rules/` 目录下的规则示例文件不处理.

## 执行流程

1. **并行扫描**: 使用多个 Agent 并行扫描不同目录, 汇总违规结果
2. **批量修复**: 对每个违规文件执行 sed 替换, 跳过 `plugins/hookify/rules/` 目录
3. **最终验证**: 再次扫描确认零违规, 输出报告

## 修复命令

```bash
sed -i '' 's/，/,/g; s/。/./g; s/、/,/g; s/：/:/g; s/；/;/g; s/【/[/g; s/】/]/g; s/（/(/g; s/）/)/g' <file>
```

## 输出报告

```
=== 标点符号检查报告 ===
扫描范围: <目录/文件>
违规文件: X 个
修复文件: Y 个
剩余违规: Z 个
```
