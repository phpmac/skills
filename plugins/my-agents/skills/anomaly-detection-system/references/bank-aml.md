# 银行/金融 AML 监控规则

## 检测场景

银行反洗钱(AML)监控系统,基于金融行业标准实践,包含CTR,SAR,Structuring检测等.

## 大额交易报告 (CTR)

### CTR阈值

| 地区 | 阈值 |
|------|------|
| 美国 | $10,000 |
| 欧盟 | €10,000 |
| 阿联酋 | AED 55,000 |
| 印度 | INR 50,000 |

### CTR触发条件

- 现金存款超过阈值
- 现金提取超过阈值
- 货币兑换(含外币)
- 现金支付可转让票据
- 账户间现金转账

### 聚合规则

- 现金存入与现金提取分开计算,不可互抵
- 同一客户在不同分支机构的交易需聚合
- 代表同一实体的不同个人的交易需聚合

---

## 可疑交易报告 (SAR)

### SAR阈值

| 机构类型 | 阈值 |
|----------|------|
| 银行及大多数金融机构 | $5,000 |
| 货币服务机构(MSB) | $2,000 |

### SAR触发条件

- 涉及非法活动所得资金
- 旨在规避BSA要求(拆分交易)
- 无明显业务或合法目的
- 与客户正常预期交易不符

---

## 拆分交易检测 (Structuring)

### 检测模式

| 类型 | 描述 |
|------|------|
| 阈值聚合规则 | 计算7/14/30天滚动累计总额,检测低于报告阈值的存款汇总 |
| 模式检测 | 标记多次略低于报告阈值的交易(如$9,500-$9,900重复) |
| 频率规则 | 检测短时间内的异常现金存款或取款频率 |
| 时间聚类检测 | 跨不同日期但在压缩时间窗口内的多笔交易 |

### 红旗指标

- 多笔略低于报告阈值的存款,持续数天/数周
- 跨不同分支,ATM或代理的小额日常交易
- 多人代表同一实体进行交易
- 结构性存款后快速转出境外
- 与客户业务活动不符的存款模式

## 客户风险评级

### 三级风险分类

| 风险等级 | 特征 | 审查频率 |
|----------|------|----------|
| 低风险 | 低风险地区,可预测交易 | 每3-5年 |
| 中风险 | 有限红旗风险(如国际交易) | 每1-3年 |
| 高风险 | PEP身份,复杂结构 | 每年或更频繁 |

### 风险评分模型

| 风险因素 | 低(1分) | 中(3分) | 高(5分) |
|----------|---------|---------|---------|
| 客户类型 | 受监管金融机构 | 标准个人/成熟企业 | 壳公司,现金密集型企业 |
| 地理因素 | FATF成员 | 中等风险司法管辖区 | FATF高风险或制裁地区 |
| 资金来源 | 已验证工资 | 部分有文档记录 | 来源不明,离岸信托 |
| PEP状态 | 非PEP | - | PEP或负面新闻 |

### 评分阈值

| 总分 | 风险类别 | 要求的行动 |
|------|----------|------------|
| 0-10分 | 低风险 | 标准CDD |
| 11-20分 | 中风险 | 标准CDD + 增强监控 |
| 21+分 | 高风险 | 增强尽职调查(EDD) |

## 交易行为异常红旗

| 红旗类型 | 描述 |
|----------|------|
| 快速资金流动 | 存款后立即取款 |
| 地理位置异常 | 不可能旅行场景(如9:00 AM在纽约,9:15 AM在伦敦) |
| 拆分交易 | 一系列旨在规避报告要求的交易 |
| 档案不一致 | 与账户历史不符的交易 |
| 渠道异常 | 使用高风险渠道或不熟悉设备 |

---

## 跨境交易监控

### 跨境监控红旗

| 红旗类型 | 描述 |
|----------|------|
| 拆分交易 | 多笔小额交易分散风险 |
| 分层 | 通过多个国家快速转移资金 |
| Smurfing | 使用多个账户分散大额交易 |
| 往返交易 | 资金返回发送方 |

## 配置示例

```yaml
aml:
  # CTR配置
  ctr:
    us_threshold: 10000
    eu_threshold: 10000
    aggregation_days: 1

  # SAR配置
  sar:
    bank_threshold: 5000
    msb_threshold: 2000

  # Structuring检测
  structuring:
    window_days: 30
    sub_threshold_ratio: 0.95
    frequency_threshold: 5
    clustering_window_hours: 24

  # 客户风险评级
  risk_rating:
    low_max_score: 10
    medium_max_score: 20
    review_frequency_days:
      low_risk: 1095
      medium_risk: 730
      high_risk: 365

  # 红旗指标
  red_flags:
    fast_money_velocity_minutes: 10
    impossible_travel_hours: 1
    channel_risk_weights:
      atm: 0.5
      branch: 0.3
      online: 0.7
      mobile: 0.6
```
